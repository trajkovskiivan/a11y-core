SkipLink

A skip link is placed at the very top of the page. Press Tab at the top of this page to reveal it. It allows keyboard users to skip past navigation and jump to the main content.

How to test: Click in the browser address bar, then press Tab. The skip link will appear at the top-left corner of the page.

The skip link at the top of this page uses: <a11y-skip-link target="#main-content">

compa11y is an accessible component library built with WCAG AAA compliance in mind. Every component ships with full keyboard navigation, ARIA attributes, and screen reader support.

Install via npm: npm install @compa11y/web or npm install @compa11y/react.

Then import the components you need and they register themselves automatically.

Yes! The @compa11y/web package provides vanilla Web Components that work with any framework or plain HTML. The @compa11y/react package provides React compound components.
