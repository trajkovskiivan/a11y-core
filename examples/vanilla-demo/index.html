A11yKit Vanilla JS Demo

These components work with just HTML and the A11yKit script. No framework required.

Dialog

A modal dialog with focus trap and keyboard support.

Confirm Delete

Are you sure you want to delete this item? This action cannot be undone.

Keyboard Navigation

  • Tab moves between focusable elements
  • Escape closes the dialog
  • Focus is trapped within the dialog

Action Menu

A dropdown menu for actions.

Keyboard Navigation

  • Enter or Space opens the menu
  • ↑ ↓ navigates items
  • Home / End jumps to first/last
  • Escape closes the menu

Tabs

Accessible tabbed interface.

General Settings

Configure your basic account settings here.

Security Settings

Manage your password and two-factor authentication.

Notification Settings

Control how and when you receive notifications.

Keyboard Navigation

  • ← → switches between tabs
  • Home / End jumps to first/last tab
  • Tab panels are automatically shown/hidden

Combobox

An autocomplete/select component with filtering.

Keyboard Navigation

  • ↑ ↓ navigates options
  • Enter selects the highlighted option
  • Escape closes the dropdown
  • Type to filter options
