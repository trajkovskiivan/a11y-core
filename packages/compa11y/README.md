# @compa11y

The complete accessibility toolkit for modern web development.

## Packages

This is a meta-package that includes everything. For specific needs, we recommend installing the individual packages:

- **[@compa11y/core](https://www.npmjs.com/package/@compa11y/core)** - Framework-agnostic primitives.
- **[@compa11y/react](https://www.npmjs.com/package/@compa11y/react)** - Accessible React components.
- **[@compa11y/web](https://www.npmjs.com/package/@compa11y/web)** - Standard Web Components (CDN ready).

## Installation

```bash
npm install @compa11y/compa11y
```
