# @a11y-core

The complete accessibility toolkit for modern web development.

## Packages

This is a meta-package that includes everything. For specific needs, we recommend installing the individual packages:

- **[@a11y-core/core](https://www.npmjs.com/package/@a11y-core/core)** - Framework-agnostic primitives.
- **[@a11y-core/react](https://www.npmjs.com/package/@a11y-core/react)** - Accessible React components.
- **[@a11y-core/web](https://www.npmjs.com/package/@a11y-core/web)** - Standard Web Components (CDN ready).

## Installation

```bash
npm install @a11y-core/a11y-core
```
